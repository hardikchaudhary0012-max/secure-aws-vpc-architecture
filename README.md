# Secure AWS VPC Architecture & Networking

## Project Overview
This is a personal project where I designed a secure AWS Virtual Private Cloud (VPC) to demonstrate cloud networking and security best practices.

## What I Did
- Designed a secure AWS VPC
- Created public and private subnets
- Configured route tables and security groups
- Implemented IAM roles and policies using least-privilege principles
- Deployed a NAT Gateway for secure outbound internet access
- Enabled AWS CloudWatch monitoring for network activity and resource health

## AWS Services Used
- Amazon VPC
- Subnets
- Route Tables
- Internet Gateway
- NAT Gateway
- Security Groups
- IAM
- CloudWatch
